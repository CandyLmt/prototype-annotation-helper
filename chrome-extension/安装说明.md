# 原型标注助手 - 安装说明

## 准备工作

Chrome 扩展需要 PNG 格式的图标文件。请按照以下步骤准备图标：

### 方法一：在线转换（推荐）

1. 打开浏览器访问：https://cloudconvert.com/svg-to-png
2. 上传 `icons/icon16.svg` 文件
3. 设置尺寸为 16x16 像素
4. 转换为 PNG 并下载，保存为 `icon16.png`
5. 重复以上步骤，创建：
   - `icon16.png` (16x16 像素)
   - `icon48.png` (48x48 像素)
   - `icon128.png` (128x128 像素)

### 方法二：使用 Preview（macOS自带）

1. 在 Finder 中打开 `icons/icon16.svg`
2. 使用 Preview 打开后，选择 "文件" → "导出"
3. 选择 PNG 格式，调整尺寸
4. 保存到 icons 文件夹

### 方法三：使用命令行（需要 ImageMagick）

```bash
cd /Users/limengting/Desktop/标注插件/chrome-extension/icons
convert icon16.svg -resize 16x16 icon16.png
convert icon16.svg -resize 48x48 icon48.png
convert icon16.svg -resize 128x128 icon128.png
```

## 安装步骤

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `chrome-extension` 文件夹
6. 扩展已安装成功！

## 使用方法

### 1. 开启标注模式

1. 点击 Chrome 工具栏的扩展图标
2. 点击 **"添加标注"** 右侧的开关按钮开启标注模式
3. 鼠标会变成十字光标，页面进入标注状态

### 2. 添加注释

1. 在标注模式下，移动鼠标到要注释的页面元素上（元素会高亮显示）
2. 点击该元素，弹出编辑框
3. 输入功能说明内容
4. 点击 **"保存"** 按钮完成添加

### 3. 查看和管理注释

1. 点击扩展图标打开弹窗
2. 点击 **"本页注释"** 区域展开注释列表
3. 在列表中可以：
   - 查看所有注释的内容、创建时间和标注者
   - 点击任意注释快速定位到页面中的对应元素
   - 使用搜索框搜索特定标注内容

### 4. 删除注释

**删除单个注释：**
1. 点击标注标记（✦图标）打开编辑框
2. 点击 **"删除"** 按钮

**批量删除注释：**
1. 在注释列表中勾选要删除的注释
2. 点击底部 **"删除选中"** 按钮

**清除本页所有注释：**
1. 点击注释数量右侧的 **"清除"** 按钮
2. 确认删除操作

### 5. 导入/导出标注

**导出标注：**
1. 点击 **"导出"** 按钮
2. 选择保存位置，标注数据将以 JSON 格式保存

**导入标注：**
1. 点击 **"导入"** 按钮
2. 选择之前导出的 JSON 文件
3. 确认导入操作

### 6. 设置用户标识

当 Chrome 无法自动获取用户信息时：
1. 在弹窗中输入用户标识（邮箱或任意名称）
2. 点击 **"保存用户标识"** 按钮

### 7. 快捷键操作

- **Esc** - 退出标注模式或关闭编辑框
- **A** - 在标注模式下重新开始选择元素

## 功能特点

✅ **无需修改原页面** - 注释以浮层形式显示，不影响原页面结构
✅ **本地存储** - 所有数据保存在浏览器本地，保护隐私
✅ **持久保存** - 刷新页面后注释依然存在
✅ **快速定位** - 点击注释可快速定位到对应元素
✅ **批量管理** - 支持查看和删除本页所有注释

## 数据存储

- 注释数据保存在 Chrome 浏览器的本地存储中
- 每个页面独立保存各自的注释
- 数据不会上传到任何服务器

```
├── background.js      # 后台脚本
├── content.js         # 页面内容脚本
├── content.css        # 标注样式
├── popup.html         # 弹窗界面
├── popup.js           # 弹窗逻辑
└── icons/
    ├── icon16.svg     # 图标源文件
    ├── icon16.png     # 16x16 图标
    ├── icon48.png     # 48x48 图标
    └── icon128.png    # 128x128 图标
```

## 常见问题

Q: 图标显示不正常？
A: 确保 icons 文件夹中有 icon16.png、icon48.png、icon128.png 三个 PNG 文件

Q: 点击元素没有反应？
A: 确保扩展已启用，点击扩展图标后再点击"添加新注释"

Q: 注释在刷新后消失了？
A: 检查是否清除了浏览器数据，确保没有清除网站数据

Q: 如何在其他页面添加注释？
A: 打开目标页面，点击扩展图标开始添加注释即可

## 技术支持

如遇到问题，请检查：
1. Chrome 扩展是否已启用
2. 开发者模式是否开启
3. 图标文件是否齐全
4. 控制台是否有错误信息
